# AI 智能测试大屏平台 - 前端功能说明

## 📊 前端概述

前端是基于 **Vue 3 + Vite + Tailwind CSS + ECharts** 构建的单页应用（SPA），提供 **6 个主要功能模块**，用于管理和监控 AI 自动化测试。

---

## 📝 2. 用例管理（Cases）

**路由：** `/cases`  
**功能：** 创建、编辑、删除测试用例（支持 Python/YAML/自然语言三种模式）。

### 你可以在这里看到：

#### 1️⃣ 用例列表（表格）
```
┌─────┬────────────┬──────┬─────────────┬─────────────┬──────┐
│ ID  │ 名称        │ 类型 │ 描述         │ 创建时间     │ 操作 │
├─────┼────────────┼──────┼─────────────┼─────────────┼──────┤
│ 1   │ 登录测试    │ Python│ 测试登录功能 │ 2026-06-05  │ 查看 │
│ 2   │ 首页导航    │ YAML │ 测试导航栏   │ 2026-06-06  │ 编辑 │
│ 3   │ 播放视频    │ 自然语言│ AI生成脚本  │ 2026-06-06  │ 删除 │
└─────┴────────────┴──────┴─────────────┴─────────────┴──────┘
```

**操作：**
- ➕ **创建用例**：点击按钮，弹出创建对话框
- 🔍 **搜索用例**：输入名称搜索
- 🔽 **筛选类型**：按 Python/YAML/自然语言筛选
- 👁️ **查看**：查看用例详情（代码/描述）
- ✏️ **编辑**：修改用例内容
- 🗑️ **删除**：删除用例

---

#### 2️⃣ 创建/编辑用例对话框
```
┌─────────────────────────────────┐
│ 创建用例                        │
│                                 │
│  名称: [登录测试        ]       │
│  类型: [Python ▼]              │
│  描述: [测试登录功能    ]       │
│                                 │
│  内容: (代码编辑器)            │
│  ┌───────────────────────┐    │
│  │ def test_login():      │    │
│  │   device.click(100,200)│    │
│  │   ...                  │    │
│  └───────────────────────┘    │
│                                 │
│  [取消]  [创建]               │
└─────────────────────────────────┘
```

**三种类型说明：**
1. **Python**：写 Python 脚本（调用 `uiautomator2` 或 `Airtest`）
2. **YAML**：写 YAML 配置（声明式测试步骤）
3. **自然语言**：用中文描述测试步骤，AI 自动生成脚本
